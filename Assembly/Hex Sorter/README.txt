DESCRIPTION
In this lab, a user can input three program arguments in two's complement format that will then be sorted and converted to decimal.
-----------
FILES
-
hex_sorter.asm

This file contains the assembly code for the lab. The code was written in MIPS.
-
test.asm

This file contains the code that allows Lab5.asm to be tested and run.

-----------
INSTRUCTIONS
This program is intended to be run using the MIPS Assembler and Runtime Simulator (MARS). Open test.asm and input three program arguments in two's complement format, then run the program.
